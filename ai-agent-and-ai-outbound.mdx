## Configure AI Agent and Run AI Outbound

<Frame>
  ![Image](/images/image-194.png)
</Frame>

Configure the AI agent first, test it, then create outbound batches with a focused prompt and recipient list.

### Configure an AI Agent

1. Open **Settings \> Agent**.
2. Select the agent you want to edit.
3. Update the AI identity, prompt, and voice.
4. Choose answer behavior for voice agents.
5. Save changes.
6. Test with WebCall before using the agent live.

### Good AI Prompt Logic

A prompt should tell the AI:

- Who it represents.
- What goal it should complete.
- What information it should collect.
- What it should not promise.
- When it should hand off or end the call.

### Create an AI Outbound Batch

<Frame>
  ![Image](/images/image-195.png)
</Frame>

1. Open **Outbound**.
2. Create a new batch.
3. Choose the calling number and AI agent.
4. Add recipients manually or by CSV.
5. Write a batch-specific greeting and prompt.
6. Test the prompt.
7. Launch.
8. Review completed, failed, and pending results.

Use outbound batches for one clear goal at a time. A reminder campaign, qualification campaign, and win-back campaign should be separate batches.
